# DevOps Production Lab

A production-like DevOps portfolio project built step by step.

## Stack

- Linux
- Bash
- Git/GitHub
- Nginx
- Docker
- Docker Compose
- GitHub Actions
- Terraform
- Kubernetes
- Helm
- Argo CD
- Prometheus
- Grafana
- Loki
- Trivy

## Project Goal

Build, deploy, monitor, secure, troubleshoot, and document a containerized API like a real production service.
